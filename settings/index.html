ReadHeizgrenzeHeating Limit
Sensor (°C) · Zeigt den aktuellen Wert direkt vom Controller (Parameter 700). Aussentemperatur, oberhalb derer der Heizbetrieb abschaltet — Warmwasser läuft weiter. Beispiel: 15 °C → WP heizt nur wenn Aussentemperatur < 15 °C. Verhindert unnötiges Heizen bei warmen Aussentemperaturen. Typisch: 15–18 °C (FBH) · 16–20 °C (Radiatoren) · 12–15 °C (gut gedämmt). Einstellbar unter Geräteeinstellungen → Temperatur-Einstellungen. Sensor (°C) · Shows the current value read directly from the controller (parameter 700). Outdoor temperature above which heating is switched off — hot water continues. Example: 15 °C → heat pump heats only when outdoor temp < 15 °C. Prevents unnecessary heating in mild weather. Typical: 15–18 °C (underfloor) · 16–20 °C (radiators) · 12–15 °C (well-insulated). Configurable under Device Settings → Temperature Settings.
ReadMax. AussentemperaturMax. Outdoor Temperature
Sensor (°C) · Konfigurierter Maximalwert der Aussentemperatur am Controller (Parameter 91). Oberhalb dieses Werts wechselt die WP in einen definierten Betriebszustand (z.B. kein Heizen, Kühlfreigabe). Einstellbar unter Geräteeinstellungen → Temperatur-Einstellungen. Sensor (°C) · Configured maximum outdoor temperature value on the controller (parameter 91). Above this value the heat pump switches to a defined operating state (e.g. no heating, cooling release). Configurable under Device Settings → Temperature Settings.
ReadWarmwasser-HystereseHot Water Hysteresis
Sensor (K) · Zeigt den aktuellen Wert direkt vom Controller (Parameter 74). Legt fest, wie weit die Warmwassertemperatur unter den Sollwert fallen muss, bevor die Nachheizung erneut startet. Beispiel: Sollwert 50 °C, Hysterese 5 K → Nachheizung beginnt bei 45 °C. Einstellbar unter Geräteeinstellungen → Temperatur-Einstellungen. Sensor (K) · Shows the current value read directly from the controller (parameter 74). Defines how far the hot water temperature must drop below the setpoint before reheating restarts. Example: setpoint 50 °C, hysteresis 5 K → reheating starts at 45 °C. Configurable under Device Settings → Temperature Settings.
ReadRücklauf-HystereseReturn Temperature Hysteresis
Sensor (K) · Nur sichtbar wenn vom Controller unterstützt (Parameter 88). Die Luxtronik regelt den Heizkreis über die Soll-Rücklauftemperatur aus der Heizkurve. Beispiel: Soll-Rücklauf 30 °C, Hysterese 3 K → WP startet bei 27 °C, stoppt bei Sollwert. Grössere Hysterese = längere Laufzeiten, weniger Starts, besserer COP. Typisch: 2–4 K (FBH) · 3–6 K (Radiatoren). Einstellbar unter Geräteeinstellungen → Heizkreis-Temperaturgrenzen. Sensor (K) · Only visible if supported by the controller (parameter 88). The Luxtronik controls the heating circuit via the target return temperature from the heating curve. Example: target return 30 °C, hysteresis 3 K → heat pump starts at 27 °C, stops at setpoint. Larger value = longer run times, fewer starts, better COP. Typical: 2–4 K (underfloor) · 3–6 K (radiators). Configurable under Device Settings → Heating Circuit Limits.

📊Wie funktioniert es?How does it work?

Da der Controller keinen elektrischen Verbrauch misst, schätzt die App anhand des Betriebszustands. Du konfigurierst einen Watt-Wert pro Status.Since the controller does not measure electrical consumption, the app estimates based on operating state. Configure a watt value per state.

  • 1Leistungssensor aktivieren und Watt-Werte konfigurierenEnable power sensor and configure watt values
  • 2measure_power (W) erscheint auf der Gerätekachelmeasure_power (W) appears on the device tile
  • 3Wenn Heizen, Warmwasser & Standby alle > 0 W: kumulierter Energiezähler (kWh) wird automatisch aktiviertIf Heating, Hot Water & Standby all > 0 W: cumulative energy meter (kWh) activates automatically
  • 4Gerät erscheint als Verbraucher im Homey Energie-DashboardDevice appears as consumer in Homey Energy dashboard
Hinweis:Note: Der kWh-Zähler wird persistent gespeichert und überlebt App-Neustarts.The kWh counter is stored persistently and survives app restarts.
